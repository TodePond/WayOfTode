## Scope
ğŸ¸ **Todes make what they need!**<br>
Make what you need (nothing more, nothing less).
```js

const add = (a, b) => a + b
```

ğŸ¤– **Robots make what they might need!**<br>
Don't waste time making what you might never need.
```js

```
